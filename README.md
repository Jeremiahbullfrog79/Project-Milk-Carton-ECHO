# ECHO: Every Child Has Options

## Interactive Safety Education Platform by Project Milk Carton

### üéÆ Overview

ECHO is a revolutionary educational game system that teaches safety awareness through multi-perspective interactive storytelling. Unlike traditional safety education, ECHO allows players to experience scenarios from three different viewpoints: Child, Parent, and Community Member.

### ‚ú® Key Features

- **Multi-Perspective Gameplay**: Experience scenarios from child, parent, and community perspectives
- **Age-Appropriate Content**: Three age ranges (11-13, 14-15, 16-17) with customized scenarios
- **ARIA AI Integration**: Intelligent hints and guidance based on real Project Milk Carton data
- **Rewind & Reflect System**: No wrong answers - every choice is a learning opportunity
- **Real Impact Dashboard**: Connects to actual resources and crisis hotlines
- **Multiple Scenarios Per Age Group**: 4 complete scenarios for each age range

### üìö Scenario Library

#### Ages 11-13 (Middle School)
1. **Lost at the Mall** - What to do when separated from guardians
2. **New Online Friend** - Recognizing safe vs unsafe online interactions
3. **The Dare** - Standing up to peer pressure
4. **Who Can I Trust?** - Identifying safe adults in different situations

#### Ages 14-15 (Early High School)
1. **The Perfect Image** - Social media exploitation awareness
2. **First Relationship** - Recognizing healthy vs unhealthy relationships
3. **Getting Home Safe** - Transportation safety and trusting instincts
4. **First Job Safety** - Recognizing legitimate employment vs exploitation

#### Ages 16-17 (Late High School)
1. **College Visit Reality** - Advanced substance and consent awareness
2. **Dream Job Nightmare** - Recognizing sophisticated labor trafficking schemes
3. **When You See Something** - Advanced bystander intervention
4. **When Your Friend Needs Help** - Supporting peers in dangerous situations

### üéØ Educational Goals

- **Critical Thinking**: Assess situations and make informed decisions
- **Communication Skills**: Learn when and how to ask for help
- **Boundary Setting**: Recognize and enforce personal boundaries
- **Pattern Recognition**: Identify manipulation, grooming, and danger signs
- **Empathy Development**: Understand how choices affect others
- **Community Responsibility**: Recognize safety as a shared obligation

### üöÄ Deployment Instructions

#### Option 1: Simple Local Server
```bash
# Navigate to the project directory
cd milk-carton-games

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (with http-server)
npx http-server -p 8000
```

Then open `http://localhost:8000` in your browser.

#### Option 2: Production Deployment

**Netlify/Vercel:**
1. Drag and drop the entire folder to Netlify/Vercel
2. No build configuration needed
3. Instant deployment

**GitHub Pages:**
1. Create a new repository
2. Upload all files
3. Enable GitHub Pages in settings
4. Access at `https://yourusername.github.io/repo-name`

**Traditional Web Hosting:**
1. Upload all files to your web server
2. Ensure `index.html` is the root file
3. No server-side processing required
